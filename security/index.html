Security

checkloops.co.uk

We run our site behind Cloudflare with HTTPS, HSTS, and strict security headers. If you find a vulnerability, please report it to security@checkloops.co.uk. Good-faith research is welcome.

Disclosure policy

  • Do not access or modify data that isn’t yours.
  • Give us a reasonable time to fix issues before public disclosure.
